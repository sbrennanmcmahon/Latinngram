Hello!

This is the page where I am developing the web app for my Latin ngram viewer. It will eventually be an interactive tool that lets you search for any Latin word (or group of words), and it will produce a graph showing how frequently the word(s) appeared over time. I have lemmatized the classical Latin literary canon, and the results are normalised to how much text survives from each decade. I expect to complete the web version by December 2024.

For now you can view the Github repo for the database and data cleaning files.

See my other projects and recent work at my personal website.
